# spatialite_sample

Sample for reading spatialite database in .NET Core 3.1 

Note: Solution platform must be set to x86 to get this project running

## Running

```
$ git clone https://github.com/bertt/spatialite_sample
$ cd spatialite_sample
$ dotnet build
$ dotnet run
```
